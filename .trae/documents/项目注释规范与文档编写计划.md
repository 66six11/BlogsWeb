# 项目注释规范与文档编写计划

## 一、项目分析

### 技术栈
- React 19 + TypeScript 5.8
- Vite 6 + Tailwind CSS 4
- React Router 7
- Zustand 5 (状态管理)
- Three.js 0.160 (3D效果)
- Tone.js + abcjs (音乐功能)
- Google GenAI (AI聊天)

### 核心功能模块
1. 3D场景和粒子效果
2. 音乐播放器和编辑器
3. AI聊天功能
4. 博客系统
5. 作品集展示
6. 响应式设计

## 二、注释添加计划

### 1. 注释规范
- **函数/方法注释**：包含功能描述、参数说明、返回值解释、异常处理
- **类注释**：包含用途和核心功能
- **复杂逻辑注释**：实现思路说明
- **关键变量注释**：含义和使用场景
- **文件头部注释**：文件用途、作者、创建日期等

### 2. 注释添加优先级

#### 核心文件
- `/src/core/App.tsx` - 主应用组件
- `/src/store/` - 状态管理文件
- `/src/components/common/` - 通用组件
- `/src/components/features/` - 功能组件
- `/src/services/` - API服务
- `/src/utils/` - 工具函数

#### 业务逻辑文件
- `/src/pages/` - 页面组件
- `/src/components/layout/` - 布局组件
- `/api/` - API路由
